# RxCP Server

[![CodeFactor](https://www.codefactor.io/repository/github/rxcp/rxcp-server/badge/develop)](https://www.codefactor.io/repository/github/rxcp/rxcp-server/overview/develop)

The API for RxCP, a control panel for rAthena servers

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
